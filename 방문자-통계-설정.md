# 방문자 통계 (하루 몇 명?) 설정하기

사이트에 **Google Analytics 4 (GA4)** 를 연결해 두었습니다.  
설정만 하면 **하루·날짜별 방문자 수**를 웹에서 볼 수 있어요.

---

## 1단계: Google Analytics 가입

1. **https://analytics.google.com** 접속 (Google 계정 로그인)
2. **측정 시작** → 계정 이름 아무거나 (예: `빡빈킹`)
3. **속성** 만들기 → 사이트 이름 (예: `내 홈페이지`)
4. **비즈니스 정보** → 개인/취미 등 선택
5. **데이터 수집** → **웹** 선택
6. **웹사이트 URL**: `https://wjdqls21c.github.io/`
7. **스트림 만들기** 완료

---

## 2단계: 측정 ID 복사

스트림 상세 화면에서 **측정 ID** 를 복사합니다.

- 형식: `G-XXXXXXXXXX` (G- 로 시작)

---

## 3단계: 사이트에 ID 넣기

1. 폴더에서 **`js/analytics-config.js`** 파일 열기
2. 아래처럼 ID 입력:

```javascript
window.SITE_ANALYTICS = {
  ga4MeasurementId: "G-여기에복사한ID",
};
```

3. 저장 후 GitHub에 올리기:

```powershell
cd "$env:USERPROFILE\OneDrive\바탕 화면\커서 공부용"
git add .
git commit -m "Enable visitor analytics"
git push
```

4. 1~2분 뒤 **https://wjdqls21c.github.io/** 로 접속해 본인이 한 번 들어가기 (테스트)

---

## 4단계: 하루 방문자 보기

1. **https://analytics.google.com** → 만든 속성 선택
2. 왼쪽 **보고서** → **획득** → **트래픽 획득**  
   또는 **보고서 스냅샷** → **사용자**
3. 오른쪽 위 날짜: **오늘** / **지난 7일** / **지난 28일**
4. **활성 사용자** = 그 기간 방문한 사람 수 (대략)

처음에는 데이터가 **24~48시간** 뒤에 잘 잡히는 경우도 있어요.  
방금 연결했다면 **실시간** 메뉴에서 본인 접속이 보이는지 먼저 확인하세요.

- **보고서** → **실시간**: 지금 접속 중인 사람 (테스트용)

---

## 주의

- **내 PC에서 index.html 더블클릭**으로 연 경우 → 통계에 **안 잡힐 수 있음** (인터넷 주소로 들어와야 함)
- ID를 비워 두면 (`""`) 통계 스크립트는 **로드되지 않음** (방문자 추적 안 함)
- 방문자 수는 **대략치**이며, 같은 사람이 여러 번 들어오면 여러 번 잡힐 수 있음

---

Cursor 채팅에 **「방문자 통계 설정 반영해서 GitHub에 올려줘」** 라고 해도 됩니다.
